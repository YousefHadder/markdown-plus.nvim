# Keymaps Reference

Default mappings use the `<localleader>` namespace in v2.0.

## List Management

### Insert mode

| Keymap | Description |
|---|---|
| `<CR>` | Auto-continue list item / split list content |
| `<A-CR>` | Continue current list item content on next line |
| `<Tab>` | Indent list item |
| `<S-Tab>` | Outdent list item |
| `<BS>` | Smart backspace on list markers |
| `<C-t>` | Toggle checkbox |

### Normal mode

| Keymap | Description |
|---|---|
| `o` | New list item below |
| `O` | New list item above |
| `<localleader>mr` | Manual renumber ordered lists |
| `<localleader>mx` | Toggle checkbox on current line |
| `<localleader>md` | Debug list groups |

### Visual mode

| Keymap | Description |
|---|---|
| `<localleader>mx` | Toggle checkbox in selection |

### List type toggling (normal + visual)

Press `<localleader>lt` then a type key. Works in both normal and visual mode; in visual mode the decision
is range-level (all-matching lines are cleared, otherwise all are converted). Blank lines are untouched;
ordered types are renumbered automatically.

| Keymap | Converted to | Description |
|---|---|---|
| `<localleader>ltu` | `- item` | Toggle unordered list |
| `<localleader>ltt` | `- [ ] item` | Toggle task list |
| `<localleader>ltn` | `1. item` | Toggle numbered list |
| `<localleader>ltN` | `1) item` | Toggle numbered paren list |
| `<localleader>ltl` | `a. item` | Toggle lowercase letter list |
| `<localleader>ltL` | `A. item` | Toggle uppercase letter list |
| `<localleader>ltp` | `a) item` | Toggle lowercase letter paren list |
| `<localleader>ltP` | `A) item` | Toggle uppercase letter paren list |
| `<localleader>ltc` | plain text | Clear list markers |

> [!TIP]
> which-key.nvim discovers these mappings automatically and shows the menu after `<localleader>lt`.
> For a picker that waits indefinitely for the type key, bind `<Plug>(MarkdownPlusToggleListPick)` yourself:
> ```lua
> vim.keymap.set({ "n", "x" }, "<localleader>lt",
>   "<Plug>(MarkdownPlusToggleListPick)", { buffer = true })
> ```

## Code block module

| Keymap | Mode | Description |
|---|---|---|
| `<localleader>mc` | Normal | Insert fenced code block with language picker |
| `<localleader>mc` | Visual | Wrap selection in fenced code block |
| `<localleader>mC` | Normal | Change code block language |
| `]b` | Normal | Jump to next code block |
| `[b` | Normal | Jump to previous code block |

## Tables (default prefix: `<localleader>t`)

| Keymap | Description |
|---|---|
| `<localleader>tc` / `tf` / `tn` | Create / format / normalize |
| `<localleader>tir` / `tiR` / `tdr` / `tyr` | Row insert-below / insert-above / delete / duplicate |
| `<localleader>tic` / `tiC` / `tdc` / `tyc` | Column insert-right / insert-left / delete / duplicate |
| `<localleader>ta` / `tx` | Toggle alignment / clear cell |
| `<localleader>tb` / `tw` / `tW` | Insert `<br>` at cursor / wrap cell to width / strip `<br>` from cell |
| `<localleader>te` | Edit current cell in a floating popup |
| `<localleader>tmj` / `tmk` | Move row down / up |
| `<localleader>tml` / `tmh` | Move column right / left |
| `<localleader>tt` | Transpose table |
| `<localleader>tsa` / `tsd` | Sort by column ascending / descending |
| `<localleader>tvx` / `tvi` | Convert to CSV / from CSV |
| `<A-h>` `<A-j>` `<A-k>` `<A-l>` | Insert-mode table navigation |
